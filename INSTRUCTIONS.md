# Тестовое задание

Это репозиторий для выполнения тестового задания на стажировку в Авито.

Подробное условие задания находится в [TASK.md](TASK.md).
Описание API — в [api.yaml](api.yaml).

---

## Требования

- Язык реализации — на ваш выбор
- Проект должен запускаться через **Docker Compose**
- Файл `docker-compose.yaml` должен находиться в **корне репозитория**
- Сервис должен подниматься командой:

```bash
docker-compose up --build
```

- Все переменные окружения в `docker-compose.yaml` должны иметь **дефолтные значения** (чтобы проект собирался без дополнительных настроек)
- Сервис должен слушать порт **8080**
- Реализуйте `GET /_info` — эндпоинт должен возвращать `200 OK` на любой запрос

---

## Проверка задания

Работы проверяются **вручную** независимо от автоматического скоринга.

Автоматические тесты запускаются через GitHub Actions (`.github/workflows/classroom.yml`).
Они стартуют `docker-compose`, дожидаются сервиса на `localhost:8080` и прогоняют e2e-сценарии.

Результат автотестов — скор от **0 до 5**.

Артефакты запуска теста доступны 7 дней.

Чтобы запустить тесты вручную, перейдите во вкладку **Actions** вашего репозитория и запустите воркфлоу **Проверочные тесты**.
